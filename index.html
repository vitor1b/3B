for ([inicialização]; [condição]; [incremento]) { declaração } COPIAR CÓDIGO Porém, como podemos ler essa linha de código? Primeiramente, retomamos ao ponto de que ele consiste em três expressões, no caso acima, inicialização, condição e incremento. Agora vamos entender o que cada uma significa. A inicialização começa com uma expressão que inicia uma variável (geralmente um contador) e define seu valor inicial. Isso acontece apenas uma vez no início. Em seguida, o for verifica uma condição. Enquanto essa condição for verdadeira, o loop continuará a ser executado. Durante a execução, o for também irá adicionar um incremento ao código e logo após será realizado um teste de verificação do novo valor. Essa estrutura se repetirá até que a condição retorne “false” (falso). Vamos agora partir de um exemplo prático: for (var i = 1; i <= 5; i++) { console.log(i); } COPIAR CÓDIGO De acordo com o exemplo acima, o código executará: Inicialização: para i = 1; Condição: enquanto i permanecer menor ou igual a 5; Incremento: acrescentará 1 a i; Teste: chama no console para exibir o valor correspondente; Atualização: a estrutura continua fazendo testes em i, até que i seja igual a 5. A partir disso, o código encerra a repetição. Nesse caso, o console exibirá o seguinte resultado:: alt text: Captura de tela do console do navegador, exibindo o resultado de números sequenciais: 1, 2, 3, 4 e 5. Abaixo, criaremos um loop para iterar através de um array: var frutas = ["Maçã", "Banana", "Laranja", "Pera"]; for (var i = 0; i < frutas.length; i++) { console.log(frutas[i]); } COPIAR CÓDIGO Aqui, o código funcionará da seguinte forma: Para i = 0 (i indica a posição da lista frutas); Enquanto i permanecer menor do que o comprimento da lista frutas; Adiciona 1 a i; Realiza o teste chamando no console o valor de i, de acordo com a sua posição na lista, exibindo o resultado de cada repetição. Operadores comuns Já entendemos a estrutura do for e agora vamos explorar os operadores de incremento que podemos utilizar no nosso código. Veremos a seguir os operadores que são comumente usados nesse contexto. ++ : Incremento Adiciona 1 ao valor inicial de cada repetição. Equivale à equação x = x + 1, portanto, não necessita inserir um número ao lado do operador. -- : Decremento Subtrai 1 ao valor inicial de cada repetição. Equivale à equação x = x - 1, portanto, não necessita inserir um número ao lado do operador. Demais operadores Existem outros operadores além dos mais comuns. Vamos conhecê-los? += : Atribuição com Incremento Adiciona um valor determinado e atribui o resultado à própria variável, por exemplo: let contador = 5; contador += 3; COPIAR CÓDIGO Agora contador passa a ter valor 8, pois foi adicionado 3 ao valor inicial. -= : Atribuição com Decremento Subtrai um valor determinado e atribui o resultado à própria variável, por exemplo: let contador = 10; contador -= 4; COPIAR CÓDIGO Agora contador passa a ter valor 6. *= : Multiplicação com Atribuição Realiza a multiplicação por um valor determinado e atribui o resultado à própria variável, por exemplo: var numero = 3; numero *= 2; COPIAR CÓDIGO Agora, numero é igual a 6, pois foi multiplicado por 2. /= : Divisão com Atribuição Realiza a divisão por um valor determinado e atribui o resultado à própria variável, por exemplo: var numero = 10; numero /= 2; COPIAR CÓDIGO Agora, numero é igual a 5, pois foi dividido por 2. Ou seja, adicionando os parênteses na função atualizaContador: function comecaCronometro(){ setInterval(atualizaCronometro(),1000); } COPIAR CÓDIGO Quando a função comecaContagem é chamada, ela tenta definir um intervalo, mas em vez de passar a função atualizaCronometro como argumento para setInterval, ela está passando o valor retornado por atualizaContador(). A menos que atualizaContador() retorne uma função outra função, isso causará um erro porque setInterval espera uma função como seu primeiro argumento. Em resumo, o setInterval precisa receber uma referência à função que deve ser chamada, não o resultado da chamada de uma função, a menos que essa chamada retorne a outra função.
